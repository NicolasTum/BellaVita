# Manual basico de usuario

## Objetivo

Club de Compras permite registrar clientes, compras, ciclos de seis stickers y premios de una tienda de ropa.

## Primer uso

1. Abrir la aplicacion.
2. Iniciar sesion con el usuario administrador inicial.
3. Cambiar la contrasena cuando el sistema lo solicite.
4. Configurar la carpeta de copias de seguridad.

## Datos reales y seguridad

La base de datos real queda en la carpeta local segura del usuario de Windows. No debe subirse a GitHub ni guardarse junto al ejecutable.

## Copias en Google Drive

Google Drive se usara como carpeta de respaldo, no como ubicacion de la base activa. Si Google Drive no esta disponible, el programa seguira funcionando y dejara una advertencia no bloqueante.
